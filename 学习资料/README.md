# 学习资料汇总
一些机器学习领域的经典学习资料.  
虽然这些资料均可在网络上找到,但许多人在入门时较为迷茫,而每本书都有其侧重点,
故将本人在学习期间接触到的认为较好的资料总结在此,希望能够帮助到后来的师弟师妹.  
**同时欢迎大家不断补充自己认为较好的资料,前人栽树,后人乘凉.**
